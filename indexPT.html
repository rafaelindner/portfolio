Projetos Recentes

Projeto 1

Calculadora

Este projeto envolve a criação de uma calculadora convencional usando JavaScript, HTML e CSS. O aplicativo permite aos usuários realizar operações matemáticas básicas e possui uma interface intuitiva. A lógica para as operações é implementada em JavaScript, enquanto HTML e CSS são usados para estruturar e estilizar a calculadora. Ele oferece uma oportunidade para aprender e praticar habilidades de desenvolvimento web.

Projeto 2

Pokédex

O projeto que criei é baseado em uma Pokédex, utilizando HTML, CSS, JavaScript e a PokeAPI. Ele permite aos usuários pesquisar por nomes de Pokémon e visualizar suas informações. O projeto combina as habilidades de desenvolvimento front-end de HTML e CSS para criar uma interface de usuário atraente, enquanto JavaScript é usado para lidar com a interatividade e a obtenção de dados da PokeAPI.
